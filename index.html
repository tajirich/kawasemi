この頃転職を考え、職業訓練校でWeb制作を学ぶことを決める。
2021年創造者デザイン専門学校Web専攻（２年間の職業訓練）終了
2021年よりWeb制作や画像処理などの仕事に就き経験を重ねながら現在に至る
未経験職種への転職は容易ではありませんでしたが、学んだことが仕事現場で役立つこと、要望に応えることができた時の達成感を日々感じながら過ごしてきました。IT技術の進歩は勢いを増す一方ですが、柔軟に対応することを心がけたいと思います。
illustorator
実務では、文字やイラスト中心のメインビジュアル、バナー、ロゴデザイン、社内向けの機関誌制作。個人的には名刺やチラシのデザイン印刷の入稿が可能。パスを使ったイラストの制作可能。
photoshop
実務では、画像の補正や加工に使用。メインビジュアル、バナー、アイコンの制作。Webサイト用の素材はPhotoshopのみで済ませることが多い。
html
小規模なサイトからコーポレイトサイトまで制作可能。
css
小規模なサイトからコーポレイトサイトまで制作可能。sassやcss設計については書籍や動画で時々勉強している。
jQuery
スライダーやライトボックス、絞り込み検索など要望に応じて実装可能。
wordpress

javascript
